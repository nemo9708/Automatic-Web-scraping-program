# Automatic-Web-scraping-program
# 본 프로그램의 목적은 웹사이트 자동 스크래핑으로
# 깃허브의 클라우드 서버를 이용한 프로그램임.

사용 방법

1. Google Cloude Console 의 qoo10-scaraper-auto 를 찾아 상단의 "계속" 클릭
2. Github -> Automatic-Web-scraping-program 에서 Setting 클릭
3. 좌측 메뉴에서 "Secrets and variables” → “Actions” 선택
4. Secret 목록이 밑에 나와 있음. 수정하고싶은 Secret 수정

   Secret 목록
   QOO10_URL : 메가와리 링크, 메가와리 링크 변경 시 필수 수정
   HIGHLIGHT_NAME : 제품명, 제품명 변경 시 필수 수정
   GMAIL_USER : 송신자 Gmail 아이디, 변경 희망 시 수정, 실제 Gmail 계정이어야함
   GMAIL_PASS : 송신자 Gmail 비밀번호, GMAIL_USER 변경 시 필수 수정, USER에 맞는 비밀번호 여야함.
   SEND_TO : 수신자 이메일 설정
